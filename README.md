## Run Project
- git clone **(https://github.com/fatmaabdelrahman/fleet-management-system.git)**
- composer update 
-create .env file
- copy .env example file in .env and set database name and password 
- run command php artisan key:generate
- php artisan config:cache
- php artisan migrate
- php artisan db:seed
- php artisan serve
- open it in browser
